# replit.md

## Overview

FairDealHub 2.0 is a modern affiliate discounts platform that connects users with verified coupon codes and exclusive deals from top brands. The application features a futuristic dark theme with neon accents, comprehensive product discovery, robust admin management capabilities, and an advanced personalized deal recommendation engine. Built as a full-stack web application, it provides both public-facing deal browsing, administrative tools for content management, and AI-driven personalized shopping experiences.

## User Preferences

Preferred communication style: Simple, everyday language.

## System Architecture

### Frontend Architecture
- **Framework**: React with TypeScript using Vite as the build tool
- **Routing**: Wouter for client-side routing with authentication-based conditional rendering
- **State Management**: TanStack Query (React Query) for server state management
- **UI Framework**: shadcn/ui components built on Radix UI primitives with Tailwind CSS
- **Styling**: Custom CSS variables with dark theme, glassmorphism effects, and neon accent colors
- **Typography**: Space Grotesk for headings, Inter for body text

### Backend Architecture
- **Runtime**: Node.js with Express.js server
- **API Structure**: RESTful endpoints with Express routes organized by feature
- **Development Setup**: Vite middleware integration for hot reloading in development
- **Build Process**: ESBuild for production server bundling, Vite for client assets

### Authentication System
- **Provider**: Replit Auth integration with OpenID Connect
- **Session Management**: Express sessions with PostgreSQL session store
- **Authorization**: Role-based access control (admin, editor, viewer)
- **Security**: HTTP-only cookies, CSRF protection, secure session configuration

### Database Design
- **ORM**: Drizzle ORM with PostgreSQL dialect
- **Provider**: Neon Database (serverless PostgreSQL)
- **Schema**: Structured tables for users, categories, merchants, products, coupons, subscribers, and clickout tracking
- **Relationships**: Foreign key relationships between entities with proper indexing
- **Migrations**: Drizzle Kit for schema management and database migrations

### Data Architecture
- **Products**: Comprehensive product catalog with pricing, ratings, and categorization
- **Merchants**: Brand information with logos and affiliate tracking
- **Categories**: Hierarchical category system with icon support
- **Coupons**: Coupon code management with verification status and expiration tracking
- **Analytics**: Click tracking and conversion monitoring for affiliate links

### Personalization Engine
- **User Preferences**: Category preferences, merchant preferences, price ranges, discount thresholds
- **Behavioral Tracking**: User interaction history with products, merchants, and coupons
- **Recommendation Generation**: AI-powered algorithm generating personalized deal suggestions
- **Recommendation Scoring**: Relevance scoring system with reason categorization
- **Saved Deals**: User favorites and bookmarking system with personal notes

### Content Management
- **Admin Interface**: Comprehensive dashboard with sidebar navigation and real-time statistics
- **AI Content Generation**: Automated product discovery and import system with daily auto-updates
- **Smart Deal Detection**: AI-powered system that generates products with best deals and working codes
- **Verification**: Manual and automated coupon code verification processes
- **User Management**: Role-based user administration and subscriber management
- **Auto-Scheduler**: Daily automated product generation and deal updates

### Performance Optimization
- **Query Optimization**: Efficient database queries with proper indexing and relationships
- **Caching Strategy**: React Query for client-side caching with configurable stale times
- **Image Handling**: Cloudinary integration for optimized image delivery and management
- **Code Splitting**: Vite-based bundle optimization with lazy loading capabilities

### Error Handling
- **Client-side**: Toast notifications for user feedback and error display
- **Server-side**: Structured error responses with proper HTTP status codes
- **Logging**: Request/response logging with duration tracking
- **Monitoring**: Error tracking preparation with Sentry integration points

## External Dependencies

### Database and Storage
- **Neon Database**: Serverless PostgreSQL for primary data storage
- **Cloudinary**: Image hosting and optimization service for product images and assets

### Authentication Services
- **Replit Auth**: OpenID Connect authentication provider for user management
- **Session Storage**: PostgreSQL-based session persistence with connect-pg-simple

### UI and Component Libraries
- **Radix UI**: Headless component primitives for accessible UI elements
- **Tailwind CSS**: Utility-first CSS framework for styling and responsive design
- **Lucide React**: Icon library for consistent iconography throughout the application

### Development and Build Tools
- **Vite**: Fast build tool and development server with HMR support
- **ESBuild**: High-performance JavaScript bundler for production builds
- **TypeScript**: Type safety across frontend and backend code

### Email and Communication
